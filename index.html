Was ist eine Web-Präsentation?

Eine Präsentation mit HTML, CSS und JavaScript.

Sie ist wie eine Webseite.

Man kann klicken und scrollen.

What is a web presentation?

A presentation made of HTML, CSS, and JavaScript.

It's like a website.

You can click and scroll.

ウェブサイト型プレゼンテーションとは？

HTML、CSS、JavaScriptで作成されたプレゼンのこと。

Webサイトと同じようなもの。

クリックしたり、スクロールしたりできる。
